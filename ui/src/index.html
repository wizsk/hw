Keyboard Shortcuts & Search Info

  • /roots verb family details.
  • The Magnifin icon is used for root search.
  • The other icon is used for searching the entire dictionary for matching phrases.
  • Note: Every shortcut will work on any keyboard (English or Arabic).
  • Press ESCAPE key to remove focus from the search input.
  • Press CTRL + Enter for full text search.
  • Press s to focus on the search bar and select all the text inside.
  • Press i to focus on the search bar and move the cursor to the end of text.
  • Press r and t to alternate between root and text search while on the search page.
  • Press f to load /roots page.
  • Press h to load home page.
  • Press j to scroll down.
  • Press k to scroll up.
